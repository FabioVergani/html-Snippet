wrap="soft" (default value)
all line breaks in the value consist of a CR+LF pair, but does not insert any additional line breaks. The Soft state indicates that the text in the textarea is not to be wrapped when it is submitted (though it can still be wrapped in the rendering).

wrap="hard"
inserts line breaks (CR+LF) so that each line has no more than the width of the control; (the cols attribute must be specified.) The Hard state indicates that the text in the textarea is to have newlines added by the user agent so that the text is wrapped when it is submitted.
